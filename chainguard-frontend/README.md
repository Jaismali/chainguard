#  ChainGuard
### Next-Gen Digital Forensics Ecosystem — Powered by Blockchain

> Cybercrime evidence in India is vulnerable to tampering before reaching court.
> ChainGuard makes tampering impossible — immutable, timestamped, role-gated.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Blockchain | Ethereum Sepolia Testnet |
| Smart Contracts | Solidity + Hardhat + OpenZeppelin |
| File Storage | IPFS via Pinata API |
| Frontend | React.js + ethers.js + MetaMask |
| Backend | Node.js + Express |
| Database | MongoDB |
| Styling | Tailwind CSS |

## ⚙️ Features

- 🔒 Immutable evidence logging via SHA-256 + IPFS + Ethereum
- 🔗 Automated chain-of-custody tracking via smart contracts
- 👤 Role-based access control (Investigator / Auditor / Court / Admin)
- ✅ Real-time verification of investigative actions
- 📋 Tamper-proof audit trail for legal admissibility
